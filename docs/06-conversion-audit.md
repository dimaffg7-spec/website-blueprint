# 06 - Конверсионный аудит лендинга

Самый ценный переносимый процесс: превратить «страница вроде норм» в диагноз с приоритетами. Проводится в **отдельном свежем чате** - тот, кто верстал, своих дыр не видит.

## Метод: 5 линз → 2 проверки → синтез

Аудировать **живую версию** (не локальную ветку - там бывает рассинхрон контента). Пять независимых экспертных линз (каждая изолированно, не зная выводов других) → две adversarial-проверки → синтез. Полный промпт - [`../prompts/copy-paste-prompts.md`](../prompts/copy-paste-prompts.md).

Линзы:
1. **StoryBrand (SB7):** проходит ли за 5 секунд «кто герой / какая его проблема / какой план / какой призыв»; не перегружена ли страница (шум).
2. **Direct Response:** таблица давления - Оффер / Выгода / Доказательства / Возражения / CTA (по каждому: статус слабо-средне-сильно + что не так + что делать).
3. **Ясность коммуникации:** понятно ли за 5 секунд, кто клиент и что ему предлагают.
4. **Потери конверсии:** пройти страницу по экранам сверху вниз `[0]..[N]` - где выпадает внимание / почему / что сделать. Привязка к реальным экранам, не абстракции.
5. **Копирайтинг:** заголовки, офферы, микрокопия.

Проверки после линз:
- (а) **Фактчек** - нет ли выдуманных цифр/обещаний;
- (б) **Соответствие бренду и голосу**.

Синтез: топ-проблемы, подтверждённые несколькими линзами одновременно.

## Что обязательно в отчёте

- Диагноз одной фразой + топ-3 проблемы (подтверждённые ≥2 линзами).
- Заголовки/офферы - **всегда 3 варианта**, не 1 («сейчас / лучше вар.1 / вар.2 / вар.3»).
- CTA раздельно: прямой (с микрокопией под кнопкой) и переходный (лид-магнит с низким барьером).
- **Не выдумывать цифры** - где данных нет, ставить пометку `[нужны данные от владельца]`.
- Сплит «делает AI / нужен человек» - два явных списка.
- Приоритеты топ-7 по «эффект × сложность».
- Трекер решений `[применить] / [отказать] / [отложить]` - правки не катятся без отмашки владельца.

## Типовые находки (проверять первым делом)

- **Первый экран без оффера и CTA** (особенно если там длинное scroll-видео) - главная точка выпадения.
- Соц-доказательство «про эмоцию, а не про результат» (благодарности вместо измеримого эффекта).
- Хайп-заголовки без доказательств рядом («окупается с первой недели») - конфликт с премиум-брендом.
- Возражение №1 целевого клиента (для B2B часто - безопасность данных) закопано глубоко.
- Слишком мало CTA на длинную страницу / один-единственный канал конверсии.
